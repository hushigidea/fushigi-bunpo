hushigi_grammar
===============================

version number: 0.0.6

Overview
--------

Assistant for aiding conversational fluency for Japanese language.

Installation / Usage
--------------------

To install use pip:

    $ pip install hushigi_grammar


Or clone the repo:

    $ git clone https://github.com/hushigidea/hushigi-grammar.git
    $ python setup.py install
